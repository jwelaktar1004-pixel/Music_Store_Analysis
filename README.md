# Music Store — SQL Data Analysis

## Overview
This project analyses a music store database (albums, artists, tracks, customers, invoices) to identify:
- Top revenue cities and countries
- Best customers and spending behaviour
- Rock genre listeners and top Rock artists
- Track length insights and genre popularity by country

## Objective
Help the business grow revenue by enabling targeted marketing, event planning, VIP programs and artist partnerships.

## Data
Tables: `album, artist, customer, employee, genre, invoice, invoice_line, media_type, playlist, playlist_track, track`.

Data cleaning performed (Python): null checks, encoding fixes, duplicate removal, type casting, FK validation.


## How to run
1. Load the database (Chinook-like schema).
2. Run `sql/queries.sql` in your SQL client.
3. Export results to CSV for visualization.
